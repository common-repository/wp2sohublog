同步发表 WordPress 博客日志到 搜狐博客,初次安装必须设置后才能使用。

== Description ==


Version 1.0.1 支持功能


1。支持选择发布到搜狐博客中文章的评论权限。


2。支持将Wordpress中文章链接发布到搜狐博客，并可选原文链接显示的位置。

3。支持将Wordpress中文章的标签发布到搜狐博客。

Version 1.0.1 不支持功能

1。不支持将Wordpress中私密(private)文章发布到搜狐博客。

2。不支持将Wordpress中的文章发布成搜狐博客中私密博客。


== Installation ==

1. 上传 `wp2sohublog.php` 到 `/wp-content/plugins/` 目录

2. 在Wordpress后台控制面板"插件(Plugins)"菜单下激活wp2sohublog插件

3. 在Wordpress后台控制面板"配置(Settings)->wp2sohublog"菜单下设置插件的必须信息。（只有经过设置，插件才能正常使用）
